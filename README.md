# PA_IEMOCAP
Personality labels of IEMOCAP dataset


tipi_labels.csv contains the original TIPI annotations of 6 annotators.

ocean.csv is the processed OCEAN labels of each speaker in IEMOCAP dataset.

The processing and other codes available at https://github.com/gitgaviny/IEMOCAP_personality
